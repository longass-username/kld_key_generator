Программа была создана для автоматической
генерации ключей по заданному шаблону.

Для корректной работы программы, выполня-
емый файл должен быть размещен в одной 
дериктории с папкой "output", куда и будут
записываться все сгенерированные файлы.
На компьюетере должны быть:
- интерпретатор Python 3.X
- библиотека PyQt5
- библиотека Cryptodome (pycryptodome)

Описание работы с программой:
- Запустите исполняемый файл, предварите-
  льно убедтвшись, что системные требова-
  ния удовлетворены.

- Введите ключ ZMK.

- Нажмите кнопку "Apply ZMK". ZMK может 
  быть отклонен программой, если библио-
  тека Cryptodome посчитает ключ слишком 
  простым.

- Выставьте необходимые настройки формата
  и количества ключей.

- Нажмите кнопку "Apply settings".

- Нажмите кнопку "START" и программа
  начнет работу.

- В текстовом поле вы увидете название 
  файла и полный путь до его расположения 
  в системе.
